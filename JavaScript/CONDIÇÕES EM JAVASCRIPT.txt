CONDIÇÕES EM JAVASCRIPT

Sequências:
Todos os códigos são sequenciais: 
var = n = 3
n += 2
window.alert(n)

As condições são como se fosse uma bifurcação com desvios condicionais
o sistema segue um fluxo, acha uma bifurcação, encontra o que quer e segue novamente.

true

} else {

false

}
